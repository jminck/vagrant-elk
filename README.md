# Vagrant ELK stack

A vagrant based ELK stack (Elasticsearch, Logstash, Kibana).

Installation Steps:

```bash
vagrant up
visit URL for kibana - http://127.0.0.1:5601
Elasticsearch - http://127.0.0.1:9200
```

## Installed versions 

Contains the following software 
- Elasticsearch 6.2.4
- Logstash 6.2.4
- Kibana 6.2.4 


## Configuration folders

```bash
# Elasticsearch
/etc/elasticsearch/

# Logstash
/etc/logstash/conf.d

# Kibana
/etc/kibana
```
